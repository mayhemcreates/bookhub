# UI Suggestions

- **Top bar hamburger** - this duplicates the functionality of the sidebar navigation on desktop. This could probably be removed. 
-**colour palette** - contrast between text and background colours should be increased. 
- **Consistent CTAs** - action buttons on cards should be consistent with other CTA. They're also fairly small and missable. 
- **Modals** - not great for accessibility - would suggest a path redirect instead
- **Search button** - where this was missing from designs, I have implemented ajax (with aria-live / aria-busy pattern to announce page changes to screenreaders), but having a search button would be more accessible.
